# НовыйДом НВ — статический сайт

Сайт про строительство домов, бань, кровельные и общестроительные работы в Нижневартовске.

Готовый многостраничный сайт для GitHub Pages.

## Что внутри
- `index.html` — главная
- `services.html` — услуги
- `portfolio.html` — портфолио
- `contacts.html` — контакты
- `styles.css` — стили
- `script.js` — мобильное меню
- `assets/hero-source.jpg` — изображение для блока на главной

## Как загрузить на GitHub Pages
1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из этой папки в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`.
6. Сохраните настройки — через минуту сайт будет доступен по ссылке GitHub Pages.
